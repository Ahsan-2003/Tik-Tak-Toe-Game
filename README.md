# Tic Tac Toe Game

A classic two-player Tic Tac Toe game built with **HTML**, **CSS**, and **JavaScript**. Play with a friend and enjoy a clean, responsive interface.

## Features

- Two-player turn-based gameplay (X vs O).
- Dynamic win/draw detection with instant result display.
- Reset button to start a new game anytime.
- Responsive design – works on desktop, tablet, and mobile.

## How to Run

1. **Clone the repository**  
   ```bash
   git clone https://github.com/your-username/tic-tac-toe.git
   ```
2. **Open the game**  
   - Navigate to the project folder and double-click `index.html`.  
   - Or use a local development server (e.g., VS Code Live Server) for the best experience.

That's it – no build steps or dependencies required!

## Technologies Used

- HTML5
- CSS3 (Flexbox/Grid, custom styling)
- JavaScript (ES6)
